# what_to_eat_today
choose what to eat today randomly

今天吃什么 🍽️
解决"今天吃什么"的终极难题！

✨ 功能
🎲 点击按钮，随机选择一道菜
🔄 趣味的滚动动画效果
📱 手机电脑都能用
🚀 在线体验
点击访问：[你的 GitHub Pages 链接]

🎮 使用方法
点击"开始选择"按钮
看着菜品滚动
得到今天的晚餐答案！

📝 修改菜品
编辑 what_to_eat.html 文件，找到这段代码：
const dishes = [
    "红烧肉", "宫保鸡丁", "麻婆豆腐",
    "糖醋里脊", "鱼香肉丝", "回锅肉",
    // ... 添加你喜欢的菜
];
把里面的菜名换成你常做的菜就可以了。

📦 项目文件
what_to_eat.html - 主程序文件
README.md - 说明文件
.gitignore - Git配置文件

💡 本地运行
双击 what_to_eat.html 即可在浏览器中使用。
